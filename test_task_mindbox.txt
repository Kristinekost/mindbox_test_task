# Тестирование формы для оформления паспорта по возрасту

## Граничные значения

1. Минимальное значение (0):
   Входное значение: 0
   Ожидаемый результат: "Нельзя оформить"

2. Значение на границе мин (1):
   Входное значение: 1
   Ожидаемый результат: "Можно оформить"

3. Значение на границе макс-1 (114):
   Входное значение: 114
   Ожидаемый результат: "Можно оформить"

4. Максимальное значение (115):
   Входное значение: 115
   Ожидаемый результат: "Можно оформить"

5. Значение за границей макс (116):
   Входное значение: 116
   Ожидаемый результат: "Ошибка"

## Классы эквивалентности

1. Возраст < 1:
   Входное значение: -5
   Ожидаемый результат: "Ошибка"

2. 1 ≤ Возраст ≤ 115:
   Входное значение: 30
   Ожидаемый результат: "Можно оформить"

3. Возраст > 115:
   Входное значение: 120
   Ожидаемый результат: "Ошибка"

4. Возраст = 0:
   Входное значение: 0
   Ожидаемый результат: "Нельзя оформить"
